Run for Windows

1. Install Python 2.7 if not already installed
2. Double click on ProgrammingAssignment4.py
	a. This should automatically start the game
	
3. If step 2 doesn't work
	a. Start Python IDLE 2.7
	b. Open ProgrammingAssignment4.py from within IDLE
	c. Press f5 or go to Run->Run Module inside of the Python GUI
4. If steps 2 and 3 don't work, my apologies, I don't know why.  But I'm attaching screen shots of my game for you


Playing on Windows

Use the mouse to move your ship and the left mouse button to fire bullets
